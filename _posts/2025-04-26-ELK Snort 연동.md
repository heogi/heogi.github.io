## Snort 로그 Elastic으로 가져오기

```bash
sudo mkdir -p /var/log/snort
sudo touch /var/log/snort/snort.log
sudo chown syslog:adm /var/log/snort/snort.log
chmod 640 /var/log/snort/snort.log
sudo systemctl restart rsyslog
```
