### TCP Syn Scan
```shell
sudo nmap -sS -p [PORT] [TARGET]
```

### Connect Scan
```shell
nmap -sT -p [PORT] [TARGET]
```

### Service Scan
```shell
sudo nmap -sV -p [PORT] [TARGET]
```
### Host Scan
```shell
sudo nmap -sn [TARGET]
```

### Host Scan 비활성화
```shell
sudo nmap -sS -Pn -p [PORT] [TARGET]
```

### 스캔 결과 저장
```shell
sudo nmap -sS -Pn -oA nmap_result -p [PORT] [TARGET]
```

### open 결과만 출력
```shell
sudo nmap -sS -Pn --open -p [PORT] [TARGET]
```

### TOP 100 포트만 스캔
```shell
sudo nmap -sS --top-ports 100 -p [PORT] [TARGET]
```

### NSE Script
```shell
#USING SMB NSE SCRIPT
sudo nmap -p 445 -Pn -n --open -sV --script "/usr/share/nmap/scripts/smb-*"
```
